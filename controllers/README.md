# Controllers
Controller board contains 30 LED-buttons. This section describes the setup and control.

## LEDs

| Button # | Comment |
| --- | --- |
| 1 - 6 | **Player 1**, Button 1 = Select |
| 8 - 14 | **Player 2**, Button 8 = Select |
| 15 | **Hotkey** |
| 15 | TBD |
| 17 - 23 | **Player 3**, Button 17 = Select |
